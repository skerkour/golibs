# gowatch

## Installation

```sh
$ go install github.com/skerkour/bloom/tools/gowatch
# or
$ git clone https://github.com/skerkour/bloom.git && cd bloom/tools/gowatch && go install
```

## How to use

```bash
$ gowatch -build="go build" -command="./myprogram"
```
